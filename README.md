# 🚗 Luxury Car Rental - Portfolio de Hajar

## 📝 Description du projet
Ce projet est une application web de location de voitures de luxe. Il permet de consulter une galerie de véhicules, d'afficher les détails techniques et de simuler une réservation. 

## 🛠️ Technologies utilisées
- **HTML5 & CSS3 :** Structure et design premium.
- **JavaScript ES6+ :** Manipulation du DOM et programmation asynchrone (Async/Await).
- **GitHub Pages :** Hébergement en ligne.

## 🚀 Fonctionnalités
- Chargement dynamique des voitures avec un indicateur de chargement.
- Gestion des erreurs avec `try/catch`.
- Fenêtre modale interactive pour les détails.
- Système de réservation fonctionnel.
